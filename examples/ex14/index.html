# Переключение между состояниями БД

В отдельных случаях очень удобно иметь два и более состояний БД и работать с этими двумя БД с помощью одного кода. Для осуществления такого желания необходимо сделать следующее:

  1. Создайте контейнер с двумя БД.
  2. Создайте два контейнера с монтированием одной и той же папки с кодом.
  3. Для этих двух сервисов в docker-compose.yml через переменную окружения укажите имя БД.
