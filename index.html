# Общая информация

Разработка ПО очень непростой процесс. Обычно нужно установить массу различного ПО. Это могут быть:

  1. Вебсервер.
  2. Сервер БД.
  3. Многое другое.

Установить весь этот набор бывает очень сложно. И единственный выход для этого попросить (нанять) админа для такой работы. С распространением docker эта проблема стала легко разрешимой.

Но бытует мнение, что docker нужен только для использования на проде. И даже там он нужен только корпорациям. Я попробую доказать, что docker может и должен использоваться на компьютерах разработчиков. И это очень просто.

Большое количество примеров можно найти на github.
